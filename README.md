# Poker d/App Python Backend

## About The Project

Python backend for NYU Blockchain and Fintech Club's Poker d/App

## Pre-requisites

Project Dependencies

- Docker
- Poetry

## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/nyublockchainfintech/poker_dapp_backend
   ```

2. Install dependencies
   ```sh
   poetry install
   ```
